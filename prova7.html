Variáveis CSS Responsivas

Esta página demonstra o uso de variáveis CSS para criar um layout responsivo

Variáveis de Tamanho de Fonte

As variáveis CSS permitem que definamos valores de tamanho de fonte que se adaptam a diferentes tamanhos de tela.

O tamanho atual do texto principal é: var(--current-font-size)

O tamanho atual do título H1 é: var(--current-h1-font-size)

O tamanho atual do título H2 é: var(--current-h2-font-size)

Variáveis de Espaçamento

Também utilizamos variáveis para espaçamentos, que se ajustam conforme o tamanho da tela.

O espaçamento atual entre elementos é: var(--current-spacing)

O espaçamento atual entre seções é: var(--current-section-spacing)
